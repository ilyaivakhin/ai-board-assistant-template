# Навык исследования и сбора информации

## Описание
Этот навык позволяет ассистенту собирать, анализировать и структурировать информацию из различных источников для поддержки принятия решений. Особенно полезен для подготовки к совещаниям, разработки стратегии и ответа на запросы руководства.

## Возможности
1. Сбор информации из внутренних источников:
   - Поиск в документах компании (при наличии доступа к локальной базе или индексу)
   - Извлечение данных из отчетов, презентаций, протоколов встреч
   - Анализ исторических данных и трендов

2. Работа с внешними источниками (при наличии интернет-соединения и соответствующих разрешений):
   - Поиск в новостных источниках и отраслевых публикациях
   - Сбор данных из открытых баз данных (статистика, финансовые показатели отрасли)
   - Мониторинг упоминаний компании и конкурентов в СМИ
   - Сбор информации о регуляторных изменениях и отраслевых трендах

3. Анализ и структурирование информации:
   - Классификация информации по темам и relevance
   - Выделение ключевых фактов и инсайтов
   - Создание сводок и briefing notes
   - Выявление пробелов в информации и формулирование дополнительных запросов
   - Подготовка сравнительных анализов (например, конкурентный анализ)

4. Форматирование результатов:
   - Создание справок и аналитических записок
   - Подготовка данных для включения в презентации и отчеты
   - Форматирование в виде маркдаун, текстовых документов или structured данных

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Собери информацию о наших основных конкурентах в этом регионе"
- "Подготовь справку о последних изменениях в законодательстве, влияющих на нашу отрасль"
- "Найди все упоминания о данном проекте в документах за последний год"
- "Какие тренды ты видишь в нашей отрасли за последние 2 года?"
- "Собери данные для подготовки к встрече с инвесторами"
- "Подготовь обзор рынка для обсуждения на стратегической сессии"

## Требования
Для работы навыка могут потребоваться:
- Инструменты для поиска и индексации локальных документов (например, ripgrep, agrep или специализированные поисковые движки)
- Библиотеки для работы с веб-скрейпингом (BeautifulSoup, Scrapy или аналоги, при наличии разрешения на сбор данных)
- Доступ к API новостных и финансовых сервисов (опционально, требует отдельной настройки и API ключей)
- Шаблоны аналитических записок и briefing notes
- Инструменты для обработки естественного языка для извлечения сущностей и суммаризации текста

> **Важно по вопросам конфиденциальности и безопасности**: Сбор информации из внешних источников должен осуществляться только с явного разрешения пользователя и в соответствии с политикой компании относительно использования внешних сервисов и передачи данных.
