# Навык создания и редактирования презентаций

## Описание
Этот навык позволяет ассистенту создавать и редактировать презентации для деловых целей:
- Создание новых презентаций из шаблонов
- Редактирование существующих презентаций
- Добавление слайдов с различным содержимым
- Вставка текста, изображений, таблиц и диаграмм
- Применение корпоративного стиля и форматирования

## Возможности
1. Создание презентации из нуля или по шаблону
2. Добавление заголовков, подзаголовков и основного текста
3. Вставка и форматирование таблиц
4. Добавление изображений с подписями
5. Создание диаграмм и графиков из данных
6. Применение тем оформления и цветовых схем
7. Настройка анимаций и переходов
8. Экспорт в различные форматы (PDF, PPTX)

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Создай презентацию на текущую стратегию компании"
- "Добавь слайд с финансовыми показателями за квартал"
- "Вставь эту таблицу в презентацию на третьем слайде"
- "Создай диаграмму роста продаж из этих данных"
- "Примени корпоративный шаблон к этой презентации"

## Требования
Для работы навыка могут потребоваться:
- python-pptx или аналогичная библиотека для работы с PowerPoint
- matplotlib, seaborn или аналоги для создания графиков
- PIL/Pillow для работы с изображениями
