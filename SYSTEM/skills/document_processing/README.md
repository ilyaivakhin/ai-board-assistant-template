# Навык обработки документов

## Описание
Этот навык позволяет ассистенту работать с различными типами офисных документов:
- PDF файлы (чтение, извлечение текста, анализ)
- Excel таблицы (чтение данных, анализ, базовые вычисления)
- Word документы (чтение, анализ структуры, извлечение ключевой информации)

## Возможности
1. Извлечение текста из PDF документов
2. Чтение и анализ данных из Excel файлов
3. Анализ содержимого Word документов
4. Поиск конкретной информации в документах
5. Создание кратких резюме документов
6. Сравнение версий документов
7. Извлечение таблиц и графиков из документов

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Прочитай этот PDF файл и выдели основные моменты"
- "Проанализируй данные в этом Excel файле"
- "Найди в этом Word документе все упоминания о рисках"
- "Сравни эти два договора и покажи отличия"

## Требования
Для полной работы навыка могут потребоваться дополнительные инструменты:
- pdftotext или аналоги для работы с PDF
- pandas или аналоги для работы с Excel
- python-docx или аналоги для работы с Word

Эти зависимости можно установить через requirements.txt или аналогичный механизм.
