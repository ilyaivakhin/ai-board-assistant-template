# ESG и устойчивое развитие

## Описание
Навык для оценки экологической, социальной и управленческой (ESG) деятельности производственной компании, анализа соответствия целям устойчивого развития и подготовки рекомендаций по улучшению ESG-показателей.

## Что анализирует
- Экологические показатели: выбросы, потребление энергии и воды, управление отходами
- Социальные аспекты: условия труда, безопасность, отношения с местным сообществом, развитие персонала
- Управление: структура совета, антикоррупционные меры, прозрачность, вовлечение заинтересованных сторон
- Соответствие национальным и международным ESG-стандартам (ГРИ, SASB, ТCFD, ЕС Таксономия)

## Как использовать
1. Поместите ESG-отчеты, данные по экологическим показателям, политики и процедуры в папку `Inputs/`
2. Запустите навык: `opencode run esg_and_sustainability`
3. Результаты появятся в папке `Outputs/`

## Выходные данные
- ESG-оценка по ключевым направлениям
- Выявленные сильные стороны и зоны улучшения
- Сравнение с отраслевыми бенчмарками
- Рекомендации по улучшению ESG-позиции компании
- План действий с целями и показателями (KPIs)
- Список использованных источников и данных

## Требования
- Python 3.8+
- Требуемые библиотеки: pandas, pdfplumber, requests (для получения внешних данных при необходимости)
