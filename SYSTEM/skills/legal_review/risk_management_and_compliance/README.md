# Управление рисками и проверка соблюдения нормативов

## Описание
Навык для выявления, оценки и мониторинга ключевых рисков производственной компании, а также проверки соблюдения отраслевых нормативов, техрегламентов и стандартов безопасности.

## Что проверяет
- Оценка операционных рисков (производственные аварии, простои, качество продукции)
- Финансовые риски (кредитные, валютные, процентные)
- Юридические риски (судебные иски, штрафы, нарушения нормативов)
- Соблюдение экологических, санитарных и трудовых нормативов
- Соответствие отраслевым стандартам (ISO, ГОСТ и др.)

## Как использовать
1. Поместите внутренние документы (политики, инструкции, отчеты по инцидентам, результаты аудитов) в папку `Inputs/`
2. Запустите навык: `opencode run risk_management_and_compliance`
3. Результаты появятся в папке `Outputs/`

## Выходные данные
- Матрица рисков (вероятность × влияние)
- Список выявленных несоответствий и зон риска
- Рекомендации по снижению рисков и улучшению системы управления
- План мероприятий с ответственными и сроками
- Список использованных источников

## Требования
- Python 3.8+
- Требуемые библиотеки: pandas, pdfplumber, docx (при необходимости)
