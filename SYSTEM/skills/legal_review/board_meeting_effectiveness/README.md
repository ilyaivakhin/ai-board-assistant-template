# Оценка эффективности совещаний совета директоров

## Описание
Навык для оценки эффективности работы совета директоров производственной компании: подготовка и проведение совещаний, качество принятых решений, follow-up и исполнение поручений.

## Что оценивает
- Подготовка материалов к совещаниям (своевременность, полнота, качество аналитики)
- Структура и дисциплина проведения совещаний (повестка, время, участие членов совета)
- Качество обсуждения и принятия решений (аргументированность, учет рисков, документирование)
- Исполнение решений и поручений (сроки, ответственность, отчетность)
- Обратная связь и улучшение процесса работы совета

## Как использовать
1. Поместите материалы совещаний (повестки, протоколы, презентации, списки поручений) в папку `Inputs/`
2. Запустите навык: `opencode run board_meeting_effectiveness`
3. Результаты появятся в папке `Outputs/`

## Выходные данные
- Оценка эффективности по каждому критерию (баллы, цветовая индикация)
- Выявленные зоны улучшения в подготовке и проведении совещаний
- Рекомендации по улучшению качества решений и follow-up
- План мероприятий по повышению эффективности работы совета
- Список использованных источников

## Требования
- Python 3.8+
- Требуемые библиотеки: pandas, pdfplumber, python-docx (для Word/протоколов)
