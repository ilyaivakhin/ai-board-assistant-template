# Навык юридической экспертизы документов

## Описание
Этот навык позволяет ассистенту проводить предварительный юридический анализ документов, выявлять потенциальные риски и обеспечивать соответствие нормативным требованиям. Особенно полезен для членов совета директоров при рассмотрении контрактов, соглашений, внутренних документов и т.д.

## Возможности
1. Анализ гражданских договоров:
   - Выявление неблагоприятных условий
   - Проверка на наличие скрытых условий и подводных камней
   - Оценка баланса прав и обязанностей сторон
   - Проверка соответствия обязательным нормам права

2. Экспертиза корпоративных документов:
   - Проверка уставов, внутренних регламентов
   - Анализ решений собраний участников/акционеров
   - Проверка соответствия корпоративным процедурам
   - Анализ конфликтов интересов

3. Работа с регуляторными требованиями:
   - Проверка соответствия отраслевому законодательству
   - Анализ требований регуляторов (ФАС, ЦБ и т.д.)
   - Оценка соблюдения антикоррупционного законодательства
   - Проверка требований по раскрытию информации

4. Выявление юридических рисков:
   - Риски оспаривания сделок
   - Риски привлечения к ответственности
   - Риски репутационного ущерба
   - Финансовые риски от неисполнения обязательств

5. Сравнительный анализ:
   - Сравнение с типовыми формами договоров
   - Сравнение с предыдущими аналогичными документами
   - Отслеживание изменений в редакциях документа

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Проверь этот договор на наличие рисков для нашей компании"
- "Соответствует ли этот документ требованиям антимонопольного законодательства?"
- "Выдели все пункты, которые могут быть оспорены в суде"
- "Сравни этот контракт с нашим стандартным шаблоном поставки"
- "Какие одобрения необходимы для сделки по этому документу?"
- "Есть ли в этом документе ограничения на конкуренцию?"

## Требования
Для работы навыка могут потребоваться:
- Библиотеки обработки естественного языка (spaCy, NLTK или аналогов для русского/английского языка)
- Шаблоны и чек-листы для типовых договоров (купля-продажа, аренда, оказание услуг и т.д.)
- Доступ к базовым нормативным актам (можно реализовать через локальную базу или API, при наличии соединения)
- Инструменты для извлечения структуры документов (выделение заголовков, пунктов, приложений)
- Шаблоны юридических заключений и меморандумов

> **Примечание**: Этот навык предоставляет предварительный анализ и не заменяет консультации квалифицированного юриста. Все выводы следует рассматривать как отправную точку для более детальной экспертизы.
