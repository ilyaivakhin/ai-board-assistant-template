# Аудит цепочки поставок

## Описание
Навык для оценки эффективности, надежности и устойчивости цепочки поставок производственной компании, включая анализ поставщиков, логистики, запасов и рисков срыва поставок.

## Что анализирует
- Надежность и квалификация ключевых поставщиков
- Уровни запасов и оборачиваемости
- Логистические сроки и стоимость транспортировки
- Риски срыва поставок (геополитические, стихийные бедствия, финансовая несостоятельность поставщиков)
- Соблюдение требований качества и стандартов поставками
- Возможности оптимизации и сокращения расходов

## Как использовать
1. Поместите данные по поставкам, договоры с поставщиками, отчеты по логистике, данные о запасах в папку `Inputs/`
2. Запустите навык: `opencode run supply_chain_audit`
3. Результаты появятся в папке `Outputs/`

## Выходные данные
- Оценка эффективности цепочки поставок по ключевым метрикам
- Выявленные узкие места и зоны риска
- Рекомендации по оптимизации запасов и логистики
- План диверсификации поставщиков и снижения зависимости
- Список использованных источников и расчетов

## Требования
- Python 3.8+
- Требуемые библиотеки: pandas, pdfplumber, openpyxl (для Excel-данных)
