# Финансовый контроль и анализ отчетности

## Описание
Навык для анализа финансовой отчетности производственной компании, оценки финансовой устойчивости, выявления отклонений от планов и подготовки рекомендаций для совета директоров.

## Что проверяет
- Анализ бухгалтерской отчетности (баланс, отчет о прибылях и убытках, отчет о движении денежных средств)
- Оценка ликвидности, платежеспособности и финансовой устойчивости
- Выявление отклонений фактических показателей от бюджета и прогнозов
- Анализ рентабельности производственной деятельности
- Оценка эффективности использования активов

## Как использовать
1. Поместите финансовые отчеты (PDF/Excel) в папку `Inputs/`
2. Запустите навык: `opencode run financial_control_and_analysis`
3. Результаты анализа появятся в папке `Outputs/`

## Выходные данные
- Краткое исполнительное резюме с ключевыми выводами
- Детальный анализ финансовых коэффициентов
- Выявленные риски и зоны внимания
- Рекомендации для улучшения финансовых показателей
- Список использованных источников и расчетов

## Требования
- Python 3.8+
- Требуемые библиотеки: pandas, openpyxl, pdfplumber (если PDF)
