# Навык подготовки протоколов и резюме встреч

## Описание
Этот навык позволяет ассистенту помогать в подготовке материалов к встречам, ведении протоколов и создании резюме после встреч:
- Создание повесток встреч на основе предыдущих решений
- Транскрибация аудиозаписей встреч (при наличии)
- Выделение ключевых решений, действий и ответственных
- Форматирование протоколов в соответствии с корпоративным стандартом
- Рассылка протоколов участникам
- Хранение и индексация протоколов для быстрого поиска

## Возможности
1. Создание повестки встречи на основе:
   - Решений предыдущих встреч
   - Текущих проектов и инициатив
   - Входящих запросов от участников
2. Обработка аудиозаписей встреч:
   - Транскрибация речи в текст
   - Выделение спикеров
   - Извлечение ключевых моментов
3. Анализ текста встречи для выявления:
   - Принятых решений
   - Назначенных действий и ответственных
   - Установленных сроков
   - Выявленных рисков и вопросов
4. Форматирование протоколов:
   - Структурированный формат с разделами
   - Выделение решений и действий
   - Приложения и ссылки на документы
5. Интеграция с календарем и задачами:
   - Создание напоминаний о действиях
   - Обновление статуса проектов
   - Создание follow-up встреч

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Создай повестку для следующего заседания совета директоров"
- "Транскрибируй эту аудиозапись встречи и выдели решения"
- "Составить протокол встречи на основе этих заметок"
- "Какие действия были назначено на последнем заседании?"
- "Напомни мне о действиях, которые должны быть выполнены до следующей недели"

## Требования
Для полной работы навыка могут потребоваться:
- Инструменты для транскрибации (например, Whisper, Google Speech-to-text, или аналоги)
- Библиотеки для работы с текстом (NLTK, spaCy или аналоги для извлечения сущностей и анализа)
- Шаблоны протоколов и повесток
- Интеграция с календарем (Google Calendar, Outlook или локальные файлы)
