# Навык анализа финансовых и стратегических документов

## Описание
Этот навык позволяет ассистенту анализировать финансовые отчеты, бюджеты, стратегические документы и другие материалы, необходимые для принятия управленческих решений.

## Возможности
1. Чтение и анализ финансовых отчетов (баланс, отчет о прибылях и убытках, отчет о движении денежных средств)
2. Анализ бюджетов иforecasts
3. Вычисление ключевых финансовых показателей (KPI): рентабельность, ликвидность, оборачиваемость и т.д.
4. Сравнение фактических показателей с плановыми и прошлыми периодами
5. Выявление трендов и аномалий в финансовых данных
6. Анализ стратегических документов: оценка рисков, возможностей, соответствия целям компании
7. Подготовка кратких финансовых резюме для руководства
8. Создание визуализаций финансовых данных (в связке с навыком визуализации данных)

## Как использовать
Чтобы использовать этот навык, пользователь может дать команды типа:
- "Проанализируй финансовый отчет за квартал и выдели основные тенденции"
- "Сравни бюджет и фактические расходы по подразделениям"
- "Вычисли ключевые финансовые коэффициенты для этого отчета"
- "Какие риски ты видишь в этом стратегическом документе?"
- "Подготовь краткое резюме финансовых результатов для совета директоров"

## Требования
Для работы навыка могут потребоваться:
- pandas, numpy для работы с табличными данными
- openpyxl или xlrd для чтения Excel файлов
- PyPDF2 или pdfplumber для извлечения текста из PDF
- matplotlib, seaborn для создания графиков (если нужна визуализация)
- statmodels или scipy для статистического анализа (опционально)
