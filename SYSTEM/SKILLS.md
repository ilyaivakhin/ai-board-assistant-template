# Список доступных скилов

- business_correspondence: Навык для составления и обработки деловой переписки.
- document_processing: Навык для обработки и анализа документов различных форматов.
- financial_analysis: Навык для финансового анализа и оценки показателей.
- legal_review: Набор навыков для юридической экспертизы и соблюдения нормативов (включает поднавыки: financial_control_and_analysis, risk_management_and_compliance, esg_and_sustainability, supply_chain_audit, board_meeting_effectiveness).
- meeting_protocols: Навык для ведения протоколов и организации совещаний.
- planning_organization: Навык для планирования и организации работы.
- presentation_creation: Навык для создания презентаций и визуализации данных.
- report_to_slide: Навык для извлечения ключевых тезисов из PDF-отчётов и (по желанию) генерации информационного слайда.
- research_information: Навык для сбора информации и подготовки исследовательских отчетов.
- visualization_data: Навык для визуализации данных и создания графиков.
